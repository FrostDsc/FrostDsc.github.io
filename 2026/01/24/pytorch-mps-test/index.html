在 M 芯片 Mac 上配置 PyTorch MPS 加速

最近入手了 M 芯片的 Mac，打算用它来跑一些深度学习的小 Demo。本文记录了如何调用 Metal Performance Shaders (MPS) 进行加速。

背景

对于统计学专业的学生来说，本地跑模型是家常便饭。相比于 TensorFlow，我个人更偏好 PyTorch 的动态图机制。

数学原理回顾

在进行张量运算前，我们简单回顾一下常用的正态分布概率密度函数：

$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

（注：如果你还没有配置 MathJax，上面的公式可能显示为源码）

代码实现

下面是一段测试代码，用于验证 MPS 是否调用成功：

import torch
import time

# 检查设备
def get_device():
    if torch.backends.mps.is_available():
        return torch.device("mps")
    elif torch.cuda.is_available():
        return torch.device("cuda")
    else:
        return torch.device("cpu")

device = get_device()
print(f"当前使用的设备: {device}")

# 创建随机张量
x = torch.randn(5000, 5000, device=device)
y = torch.randn(5000, 5000, device=device)

# 矩阵乘法测试
start_time = time.time()
z = torch.matmul(x, y)
print(f"耗时: {time.time() - start_time:.4f} 秒")
